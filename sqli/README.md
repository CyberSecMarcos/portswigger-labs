## Covered SQL Injection Techniques

- Error-Based SQL Injection

- Time-Based Blind SQL Injection

- Data extraction via inference

---

## General Methodology

The labs and scripts in this directory follow a common methodology:

1. Identify user-controlled input reflected in backend SQL queries
2. Confirm SQL injection vulnerability
3. Determine database behavior and limitations
4. Extract data using appropriate SQL injection techniques
5. Automate exploitation when possible

---

## Purpose

These scripts are intended to:
- Practice real-world SQL injection techniques
- Improve automation skills using Python
- Understand how different DBMS handle SQL injection
- Document practical security knowledge
